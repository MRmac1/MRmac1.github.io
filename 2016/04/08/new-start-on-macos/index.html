mac重装纪要

发表于 2016-04-08   |   分类于 environment

前提概要：升级了下系统，开不了机。拿去苹果店，全盘格式化了系统，啥都木有了，勉强恢复过来一些个人资料。哭…
下面对系统的恢复过程做下总结，方便下次重(千)装(万)系(不)统(要)时候查看。

个人常用资料总结

  • IED类：pycharm，webstorm，phpstorm，xcode(其中的commandLine工具很重要)
  • 设计类：sketch(UI交互设计)，sketch Toolbox(sketch的插件管理)，sip(拾色器)，principle(动画设计)，mindNode(思维导图)
  • 数据工具：Robomongo(mongodb查看)，sequel pro，DataGrip
  • 其他：Flux(屏幕色差自动调节，预防近视)

常见工具重新安装完成之后只是完成了长征的第一步。下面开始系统环境的恢复工作。

要想有个好的开发效率，有个好的终端环境是必不可少的，而且很多系统环境的安装需要依靠终端。

终端环境恢复

  • homebrew 安装
  • oh-my-zsh 安装
php环境恢复

php环境搭建包括 php+mysql+Nginx+php-fpm，其中php和php-fpm是系统自带的，所以我们只需要配置好就行了。

  • nginx安装

  • mysql 安装

或者去mysql官网下载dmg文件，用dmg文件安装的时候注意最后一步mysql会给你默认的随机密码，一定要记住。修改密码时候进入mysql终端的命令行，用随机密码登录，输入set password(‘123123’)即可。

  • 至于php-fpm和nginx配置问题可以参考这篇文章，常用命令是sudo nginx; sudo nginx -s reload; sudo php-fpm
nodejs环境恢复

nodejs环境安装就比较简单，直接去官网下载，安装好之后就有node和npm环境了。下面是几个常用的npm的工具

  • hexo nodejs的博客管理工具，使用hexo搭建的博客非常便于管理
  • phantomjs 识别那些页面上使用js生成的文本，目前我是搭配pyspider使用
  • supervisor nodejs服务器开发的利器
  • pm2 生产环境版本的nodejs部署工具，监控很完全
Python环境恢复

Python环境mac自带的就有，2.6版本和2.7版本。3.x版本的就暂时不掺和了。

  • 安装Python的包管理工具pip

  • 安装pyspider

虚拟开发环境的配置

使用虚拟开发环境主要是有些环境不想装在本机，而现成集成好的开发环境使用起来又非常方便。虚拟技术我主要使用了vagrant和docker

  • 安装vagrant dmg
  • 安装virtual box
  • 安装docker(目前还未安装)
