前言

经过nodeJS自2009年被天才程序员Ryan Dahl开源以来，JS这项古老的前端技术便在服务器端野蛮的发展。适逢CommonJS标准的释出，node实现了commonJS的大部分标准，非常完美的适应了其在后端的发展。

npm一直被标榜为node的秘密武器，确实，仓库里有25万的开源仓库，对js开发者是一笔巨大的财富了。在写后端代码时候，你只需要在本地npm install repo，然后在代码里require(‘repo’)就可以完成。确实是非常的舒服。

JS前端在前端发展又怎么样呢？在前端的发展也是风起云涌，前端页面质量要求越来越高，前端项目再也不是简简单单的HTML+jQuery+css便可以一劳永逸的了。界面的不断革新会导致前端实际上的工作量比后端要大。这时候emberJS，angularJS和React的相继出现导致前端开发的大变革。但是前端JS和后端JS始终是分置在HTTP的两端，但是前面介绍过的那么多npm模块，我们可不可以拿过来直接用呢？答案是肯定的，因为都是JS语言，并不存在兼容性问题，问题的关键在于模块加载方式。

那我们就来看看前端的模块加载方式。

2016.4.23晚间记

发表于 2016-04-23   |   分类于 随笔   |  

今天算是个平淡无奇的四月周六晚上吧。本来计划去广州和妈妈见下面，可是又被她那边坑爹的安排给耽误了。所以还是在现代城这里呆了一天。

工作两年了，经历了这么多，真的渴望能够安稳下来，可是现在安稳的局面又让自己感到有点绝望，怎么说呢？感觉自己在为了工资耗费生命。

我在想我出来工作到底是干嘛来了，自从高中毕业后，就一直在外漂泊，上海，广州，深圳，越来越觉得没有一点稳定感，至于爱情，那便是一种奢求了。

钱放在支付宝里对我来说永远都只是一个数字，两万三万的，真的没区别。室友涨了工资让我嫉妒，我并不是羡慕工资，我只是羡慕他被他的老板认可。而我，我只需要一点认可而已，当年会抛弃上海安逸的生活选择去广州，我想我也是一个能够吃苦的人，最后的离去，我想我只是没有得到应有的尊重和认可吧。商人的世界观，我这个从小地方出来的人看不透，我觉得人为了自己的目的可以欺骗别人，影响别人一生的轨迹。换做是我，我做不到。

最近一段时间我在纠结要不要回武汉，我最主要考虑的就是我爸妈。说起来，已经在外飘了六七年了，回家对我来说是一个好的选择，比之荣誉财富满身再回到故乡却父母不在，这个选择要好得多。至于我自己的职业规划，在友宝，我算是一个不错的程序员，可是公司现在对技术重视程度远远不够。我得做到经理那个位置才会有发言权，我想我等不了那么久。深圳是个不错的地方，但是我不想在这里久居，走了也不会后悔。

至于现在的室友，我想我还是会有点不舍的。除了曾杰是我同学以外，其他的人都是我到广东后才认识的。时间不长，但有的也算能够交心的了，我回去的话，应该是很难再有机会相聚了。也许会有点不舍吧。缘起缘灭的，还是走着看吧。

2016.4.21早晨的脑洞

发表于 2016-04-21   |   分类于 随笔   |  

“福利社里面什么都有，就是口袋里没有半毛钱..”，我哼唱着幸福的《童年》，我也想起了我的童年。印象中暖暖的童年回忆，却在脑海中只有些许简单的片段。无论我如何努力，也想不起一些那些曾经自己特别宝贝的记忆。

“人们许久不去回想的记忆就会丢失。就像机器一样，许久不用的内存，机器也会用垃圾回收策略把内存清理出来。也许人和机器本就一样？”。我想起了不久以前在博客上看到一本书叫《人是机器》，是一位外国友人（法国J.O.拉美特里）在1747年写成的。“人是机器”，这么简单的四个字，使用的却是祈使语气。

人的器官会损害，机器的硬件会故障，一个是去医院出钱治疗，一个是购买硬件更换。同样是要出钱。

人有智力思维，机器现在也会深度学习。而且AlphaGo的表现丝毫不比人类差，而且还是处在Alpha阶段，未来不敢想象。

就在人们常说的，机器和人的核心差距是在情感上面。

事情真的是这样的吗？

就现在来说，人和人会相互交往，是靠长得是否好看和性格品行如何。

若是机器做到了又好看，又会心疼照顾人，性格品行都没的说。那时候又会如何呢？

但也许，人现在就是机器。
